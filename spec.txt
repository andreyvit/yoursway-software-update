Для текущего приложения нужно знать:

1) список файлов с их хешами

2) версию

3) URL update-сервера

Информация о версии приложения вкючает в себя:
- список файлов с их путями и хешами;
- информация о том, какие файлы принадлежат апдейтеру;
- чтото для пользователя (название, описание и т.д.)
- дата релиза!

Теперь что касается моделек.





1) по текущей версии получить последнюю версию
2) по двум версиям получить HTML со списком изменений
3) по версии получить ее файлы
